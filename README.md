## 🚀 What OT-CRM does

OT-CRM is a customer management app for daily work with contacts, deals, and follow-up tasks. It helps you keep customer details in one place and follow the steps of each conversation with less effort.

Use it to:

- Store customer records
- Track sales contacts
- Check deal status
- Record notes and follow-ups
- Work with payment tools like Mercado Pago and PayPal
- Keep a clear view of customer activity

The app is built to support day-to-day use on Windows and to keep working through updates over time.
